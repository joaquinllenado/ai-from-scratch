# Welcome to AI from scratch!

## My goal
My goal for this repo is to learn the entire stack of building AI applications from the ground up and learning from first principles. I'm mainly doing this just to learn, so don't expect anything mindblowing or anywhere near optimal lol. 

I'm a heavy believer in project-based learning and I think if you can build it yourself (even just a crappy version), you'll gain a deeper understanding and appreciation of the advancements of this work.

If you want to follow along, I'll be posting my learnings on [X](https://x.com/joaquinllenado), [LinkedIn](https://www.linkedin.com/in/joaquinllenado/), and if I feel like it, [Instagram](https://www.instagram.com/joaquinllenado), and [TikTok](https://www.tiktok.com/@joaquinllenado).
